The onboard computer can ALSO be used throught the keyboard.


  • MAIN MENU's LAYOUT ON THE KEYBOARD
    • 5 = FLIGHT CONTROL DRIVE;
    R = ONBOARD DEVICES;
    D = PREFERENCES;
    X = DISABLE DISPLAY.

  • SUBMENUS LAYOUT
    • 6 = FIRST OPTION IN SUBMENU;
    7 = SECOND OPTION IN SUBMENU;
    8 = THIRD OPTION IN SUBMENU;
    9 = FOURTH OPTION IN SUBMENU.

  • AND SO, HERE ARE POSSIBLE KEYBOARD SHORTCUT SEQUENCES:
    • 56 - SET REMOTE TARGET;
    57 - START VIMANA FLIGHT;
    58 - SET LOCAL TARGET;
    59 - DEPLOY SURFACE CAPSULE;

    R66 - STARFIELD AMPLIFICATOR;
    R67 - LOCAL PLANETS FINDER;
    R68 - DRIVE TRACKING MODE;
    R69 - FORCE RADIATIONS LIMIT;

    R76 - INTERNAL LIGHT ON/OFF;
    R77 - REMOTE TARGET DATA;
    R78 - LOCAL TARGET DATA;
    R79 - EXTERNAL ENVIRONMENT DATA;

    R86 - LABEL STAR AS.../REMOVE STAR LABEL;
    R87 - LABEL PLANET AS.../REMOVE PLANET LABEL;
    R88 - SHOW TARGETS IN RANGE;
    R89 - SET TARGET TO PARSIS;

    R96 - RESET ONBOARD SYSTEM;
    R97 - SEND HELP REQUEST;
    R98 - SCOPE FOR LITHIUM;
    R99 - CLEAR STATUS;

    D6 - AUTO SCREEN SLEEP ON/OFF;
    D7 - REVERSE PITCH CONTROLS;
    D8 - MENUS ALWAYS ON SCREEN;
    D9 - DEPOLARIZE.
